Objetivos del Proyecto

  • Metodología reproducible para identificación automatizada de referencias culturales y musicales
  • Pipeline de procesamiento OCR/HTR para documentos históricos
  • Prompts especializados para extracción de información cultural
  • Validación estadística de la precisión de resultados
  • Interfaces de visualización para investigación humanística
  • Protocolos de reproducibilidad para futuros estudios
